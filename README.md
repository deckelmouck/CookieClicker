# CookieClicker

simple Cookie Clicker application

* Click button to generate one Cookie
* Label to show actual amount of Cookies
* several multipliers
  * 10x
  * 25x
  * 50x
  * 100x
  
### environment

Visual Studio 2019 Community Edition
Windows Forms App (.NET Core)
